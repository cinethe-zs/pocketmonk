PocketMonk

A fully private AI assistant that runs entirely on your Android device. No cloud. No subscription. No data ever leaves your phone.

100% On-Device No Internet Required Android 9+

Install

1

Download the APK

Go to the GitHub Releases page and download pocketmonkv1.0.0.apk.

2

Allow unknown apps

On your Android device, go to Settings → Security → Install unknown apps and allow your browser or Files app to install APKs.

3

Install

Open the downloaded APK from your Downloads folder and tap Install.

4

Download a model

Launch PocketMonk. On first run, the setup wizard will prompt you to download an AI model (2–4 GB depending on your choice).

⚠️ Storage space Make sure you have at least 3–5 GB of free storage before downloading a model. The model file stays on your device and is never uploaded anywhere.

First launch

The first time you open PocketMonk, a setup wizard asks you to download a model. You'll see three options:

Gemma 3 4B Recommended

Google's best small model — excellent at conversation, reasoning, and coding.

~2.5 GB

Phi-4 Mini 3.8B Fast

Microsoft's fast, powerful model — great for logic and structured tasks.

~2.5 GB

Qwen 2.5 3B Light

Alibaba's lightest model — minimal RAM usage, fastest speed, good for basic Q&A.

~1.9 GB

Tap Download on the model you want. The download bar shows real-time progress. Once complete, the app opens the chat screen.

You can download more models later from Settings → Download more models.

Chatting

Type your message in the input bar at the bottom and tap the ↑ send button (or press Return). Responses stream in token by token. The app bar shows "thinking…" while the model is generating.

Suggestion chips

On an empty conversation, four suggestion chips appear. Tap one to instantly send that prompt.

Stop generation

While the model is generating, the send button becomes a red ■ stop button. Tap it to stop generation mid-stream.

Markdown support

Assistant replies render full Markdown: headings, bold/italic, bullet lists, numbered lists, code blocks (with syntax highlighting border), blockquotes, and tables.

Voice input

If your device supports speech recognition, a 🎤 mic button appears to the left of the text field.

1

Tap the mic

The button pulses and the placeholder changes to "Listening…"

2

Speak

Your words appear in the text field in real time as you speak.

3

Send

Listening stops automatically after a short pause, or tap the mic again to stop manually. Edit the text if needed, then tap send.

📝 Note Voice input uses Android's built-in speech recognition (on-device or Google, depending on your phone settings). PocketMonk itself does not send audio anywhere.

Message actions

Long-press any message bubble to open the action menu.

ActionAvailable onWhat it does
⭐ Star / Unstar Any message Marks the message with a star indicator for reference
✏️ Edit Your messages only Opens an edit dialog — saving re-sends from that point, replacing subsequent messages
📋 Copy Any message Copies the full message text to clipboard

Regenerate

A ↻ regenerate button appears below the last assistant message (when generation is complete). Tap it to get a new response to the same prompt.

Conversations

Tap the ☰ menu icon in the top-left to open the conversation drawer.

➕

New conversation

Tap "New conversation" button or the + icon in the app bar.

🔍

Search

The search bar filters by title and message content across all conversations.

👆

Switch

Tap any conversation in the list to load it.

🗑️

Delete

Swipe a conversation left to delete it.

Conversations are automatically titled from your first message and saved to your device storage.

Tags

Tags let you organise and filter conversations.

Adding a tag

Long-press a conversation in the drawer to open the tag sheet. Type a tag name and tap Add (or press Return). Tags appear as coloured chips under the conversation title.

Removing a tag

Long-press the conversation again and tap the ✕ on any tag chip to remove it.

Filtering by tag

A row of tag chips appears above the conversation list when tags exist. Tap a tag to filter; tap All to clear the filter.

System prompt

The system prompt tells the AI who it is and how to behave. PocketMonk has a default prompt, but you can override it per conversation.

1

Tap the ⊞ tune icon

In the app bar, tap the sliders icon (it turns blue when a custom prompt is active).

2

Write your prompt

Examples: "You are a French tutor. Correct my grammar." or "You are a senior Python engineer. Be terse and technical."

3

Save

Tap Save. The prompt applies from the next message onwards. Tap Reset to default to revert.

Export a conversation

Tap the share icon (↑ box) in the app bar to copy the entire conversation to your clipboard as Markdown. You can then paste it into any notes app, email, or document.

Managing models

Tap the ⚙️ settings icon to open Settings.

Active model card

Shows which model is currently loaded.

Downloaded models

Lists all GGUF models found on your device. For each one:

  • Radio button — tap to switch to that model (reloads the AI engine)
  • Trash icon — permanently delete the model file

Download more models

Shows catalogue models not yet downloaded. Tap Download to start; a live progress bar appears. Tap Cancel to abort. If a download fails, tap Retry.

Dual-model quick-swap

You can keep two models ready and switch between them with a single tap — useful for comparing responses or using different models for different tasks.

1

Set a secondary model

In Settings → Downloaded models, tap the radio button on a non-active model while a different model is already active. The selected model becomes the secondary.

2

Use the swap chip

A small chip appears in the app bar title showing Primary ⇄ Secondary. Tap it to instantly swap — the model reloads in the background.

GPU settings

PocketMonk can offload transformer layers to the GPU for faster inference. This is experimental — not all devices support GPU inference.

SettingEffect
0 layers (default)CPU-only — stable, works on all devices
1–10 layersPartial GPU offload — modest speed gain
Full offloadMaximum GPU use — may fail on devices with limited GPU RAM

If GPU inference fails, PocketMonk automatically falls back to CPU and shows an amber warning icon in the app bar. Tap it to see a notification.

💡 Recommendation Leave GPU layers at 0 unless you experience slow generation and want to experiment. On the Pixel 7a, CPU inference runs at 5–10 tokens/second.

Performance tips

TipImpact
Close background apps before chatting Frees RAM — prevents OOM crashes during long responses
Use Qwen 2.5 3B if Gemma feels slow ~30% faster, lower RAM at cost of some quality
Keep conversations short or start new ones Context window fills up over long chats, slowing generation
Plug in your charger during heavy use Inference drains battery fast (~50% per 90 min active)
Avoid using the phone while generating Android may throttle CPU when other tasks compete

Privacy

🔒

No cloud

The AI model runs entirely on your device. No prompts or responses are ever sent to a server.

📁

Local storage only

Conversations are saved as JSON files on your device. You can delete them at any time from the drawer.

📵

Works offline

After the model is downloaded, PocketMonk works with no Wi-Fi or mobile data.

🌐

Internet only for downloads

Network access is used exclusively to download model files from Hugging Face.

FAQ

The model download keeps failing

Check your internet connection and available storage (you need ~3–5 GB free). Tap Retry in the download card. If it still fails, try a different Wi-Fi network — Hugging Face can be slow on some connections.

The app is slow / responses take a long time

LLM inference is CPU-intensive. On a Pixel 7a, expect 5–10 tokens per second. Close other apps to free RAM, and consider switching to the Qwen 2.5 3B model for faster responses.

The AI gives wrong or outdated answers

PocketMonk runs entirely offline — it has no access to the internet or real-time data. Its knowledge is limited to what was in the model's training data. For current information (news, weather, etc.) you'll need an online tool.

Can I add my own model?

Yes — copy any .gguf file to the models folder on your device: Android/data/app.pocketmonk/files/models/. It will appear automatically in Settings → Downloaded models.

Where are my conversations stored?

Conversations are saved as JSON files in the app's private documents directory. They are not accessible from the Files app but are included in Android backups.

How do I delete all my data?

Go to Android Settings → Apps → PocketMonk → Storage → Clear data. This removes all conversations and preferences. Model files can be deleted from Settings inside the app, or via the Files app.
