# PocketMonk

> Fully offline, private AI assistant for Android.
> No cloud. No subscriptions. No data ever leaves your device.

PocketMonk runs powerful language models entirely on your phone. Everything — inference, document processing, voice recognition — happens locally. Nothing is sent to any server.

**Target hardware:** Google Pixel 7a (Tensor G2, 8 GB RAM) · Android 9+ · arm64

---

## Quick start

1. Download the latest APK from the [Releases page](https://github.com/cinethe-zs/pocketmonk/releases/latest)
2. On Android: **Settings → Security → Install unknown apps** → allow your file manager
3. Open the APK and tap **Install**
4. Launch PocketMonk — on first run, select and download a model
